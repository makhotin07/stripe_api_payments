# Проект с оплатой 

### Django + Postgres + Docker + Stripe

## Клонировать проект

    $ cd project_stripe_docker

    $ cd infra

    $ docker-compose up


* Миграции происходят автоматически 
* Товары добавляются автоматически 
* SuperUser создается автоматически
* Карта для тест оплаты 

    
    $ 4242 4242 4242 4242 
    10/32 
    789
